# Chemins du site PB BZH Concept

## Domaine officiel

Le site officiel est :

https://www.pb-bzh-concept.fr

Le catalogue logiciels est :

https://www.pb-bzh-concept.fr/softwares

Le sous-domaine softwares.pb-bzh-concept.fr n'est plus utilisé comme référence principale.

## Structure publique des logiciels

Format officiel :

https://www.pb-bzh-concept.fr/softwares/<category>/<product>/<file>

Exemple :

https://www.pb-bzh-concept.fr/softwares/msi-software-packager/GestionPlanningPersonnelMobile/GestionPlanningPersonnelMobile_2.0.0.apk

## Structure locale Wamp

Racine locale :

C:\wamp64\www\pb-bzh-concept.fr\public\softwares

Format local :

C:\wamp64\www\pb-bzh-concept.fr\public\softwares\<category>\<product>\<file>

## Règle importante

WebRemoteSite doit toujours valoir :

https://www.pb-bzh-concept.fr/softwares

Il ne faut pas ajouter une deuxième fois /softwares lors de la génération des URLs.
